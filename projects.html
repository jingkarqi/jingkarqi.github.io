OCR-Translate-Test

基于多模态大模型的OCR翻译处理工具

功能特性

  • 支持图片OCR文字识别
  • 支持多模态大模型翻译
  • 支持JSON格式结果输出

安装指南

pip install -r requirements.txt

使用示例

# 执行图片处理
python pic-response.py input.jpg

# 执行批量翻译
python translator.py

项目结构

                    
GITS/
├── input_images/        # 原始图片目录
├── json-result/         # 图片处理结果
├── translated-result/   # 翻译结果
├── config.py            # API配置
├── pic-response.py      # 图片处理主程序
├── translator.py        # 翻译程序
└── qwen_vl_extractor.py # 图片处理核心模块
